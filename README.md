# FootballGuru

A web app for predicting soccer match results during major tournaments.

Features:
- Simple leaderboard with scores (3 points correct result, 1 point correct outcome)
- Anonymous login (no email needed)
- Hosted on GitHub Pages
- Real-time updates via Firebase

Planned tournaments: AFCON, Euros, World Cup
